# AskNiti

AskNiti is a Node.js based web application that helps people discover Indian government schemes relevant to them.

Many government schemes exist, but most people do not know which ones apply to them or how to understand them.
AskNiti uses AI to search, filter, and explain schemes in simple language.

The application accepts voice input from users, understands their needs (loans, farming support, education, subsidies, etc.), and returns relevant government schemes with clear explanations.

## Features
- Voice based interaction
- AI powered scheme discovery
- Simple explanations of government schemes
- Personalized scheme recommendations
- Audio responses

## Tech Stack
- Node.js
- Express.js
- AI / NLP
- Speech to Text
- Text to Speech
